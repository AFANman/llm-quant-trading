### 达利欧的核心评论

#### ✅ 做对了什么（值得保留的共识）

1. **波动率调整（Vol-Scaling）是真正的alpha来源**
   - 策略7（ATR风险平价）、策略8（Regime仓位）、策略12（收缩突破）都做到了这一点
   - 这符合我的核心理念：**风险贡献比收益预测更重要**
   - 在所有市场环境中，波动率归一化都能让策略存活更久

2. **Regime过滤（市场状态识别）是正确的方向**
   - 策略8和12用布林带宽度/波动率收缩来识别市场状态
   - 这是"全天候"思想的简化版：不同环境用不同策略
   - 但问题是：**Regime判断本身容易过拟合**

3. **动态止损（ATR-based）比固定止损好**
   - 几乎所有策略都用了ATR止损，这是正确的
   - 固定百分比止损在不同波动率环境下表现差异巨大

#### ⚠️ 我担心的问题（过度工程化风险）

1. **参数过多 → 过拟合风险极高**
   - 策略7有5+个可调参数（vol_target, fast_ma, slow_ma, regime_ma, regime_threshold...）
   - 策略8有6+个条件分支（narrow/expanding/crisis × 不同仓位比例）
   - **达利欧原则：一个策略如果需要对5个以上参数做walk-forward优化，它大概率在拟合噪声**
   - 真正robust的策略，参数应该少到"用常识就能解释"

2. **条件过多 → 样本外退化**
   - 策略3有4个入场条件（BB + MA + RSI + 交叉），策略6有5个
   - 每增加一个条件，你过滤掉了一些噪声信号，但也过滤掉了一些真实信号
   - **条件越多，样本内表现越好，样本外表现越差 — 这是过拟合的经典特征**

3. **仓位动态调整的隐性成本**
   - 策略1/6/8/12都有"根据regime调整仓位"的逻辑
   - 频繁调整仓位 = 频繁交易 = 滑点+手续费侵蚀
   - **简单持有50%仓位，在很多情况下优于频繁调仓**

4. **止盈（take_profit）的反Robustness**
   - 策略9设置了2.5倍ATR止盈，策略10/11设置了1.5%/2%止盈
   - 止盈会截断趋势的尾部收益，而趋势跟踪的核心利润恰恰来自尾部
   - **达利欧原则：不要限制上行收益，只控制下行风险**

### 达利欧的总结判断

> "这12个策略中，**策略7（风险平价）和策略8（Regime）的理念最接近全天候思维**，但它们都犯了一个错误：把'聪明'当成了'robust'。真正的robust不是参数精妙、条件精细，而是**简单到不可能过拟合**。桥水的核心策略从1990年代至今几乎没有改变过参数——因为它们的参数是'常识'，不是'优化结果'。"

## 二、达利欧策略：极简双均线趋势跟踪 + ATR动态止损

### 设计哲学

```
"分散化是唯一的免费午餐。简单是唯一的免费保险。"

- 参数数量：3个（MA20, MA60, ATR倍数=2）—— 全部是市场常识值
- 入场条件：2个 —— 少一个不够，多一个是噪声
- 出场条件：1个 —— 只控制下行，不限制上行
- 仓位：固定50% —— 不加杠杆，不做动态调仓
- 止损：2*ATR —— 给市场足够呼吸空间，同时限制最大亏损
```

### 为什么是MA20和MA60？

- **MA20**：约1个月交易日，捕捉短期趋势
- **MA60**：约1个季度交易日，捕捉中期趋势
- **这两个参数的比值（1:3）是经典配置**，不是优化出来的
- 如果MA20>MA60，说明短期趋势强于中期趋势 → 上升趋势确立
- 如果close>MA20，说明价格站在趋势之上 → 趋势仍在运行

### 为什么固定50%仓位？

- 50%仓位意味着：即使策略完全失效，最大亏损也被腰斩
- 不做动态调仓 = 零调仓成本 = 零调仓时机判断错误
- **达利欧原则：仓位管理的目的不是最大化收益，而是确保你能活到下一次机会**

## 三、达利欧的最终建议

### 组合建议

如果要将这13个策略组合成一个"全天候策略组合"，我建议：

| 策略 | 权重 | 理由 |
|------|------|------|
| 策略7（风险平价趋势跟踪） | 20% | 理念最好，波动率调整是真正的alpha |
| 策略8（Regime策略） | 15% | Regime过滤有价值，但复杂度高 |
| **策略13（达利欧极简双均线）** | **20%** | **最robust，作为组合的压舱石** |
| 策略12（波动率收缩突破） | 15% | 收缩突破逻辑清晰，止盈可去掉 |
| 策略1（RSI反转v2） | 10% | 均值回归与趋势跟踪互补 |
| 策略4（海龟突破） | 10% | 经典策略，长期验证 |
| 策略9（ATR动态布林带） | 10% | 布林带趋势跟踪有独立价值 |
| 其余5个策略 | 0% | 过拟合风险过高，建议淘汰 |

### 核心原则

> **"一个由7个低相关策略组成的组合，远好于一个精心优化的单策略。"**
> — Ray Dalio

每个策略贡献的不是收益，而是**在不同市场环境下的风险对冲**。趋势跟踪在趋势市场赚钱，均值回归在震荡市场赚钱，波动率策略在极端市场赚钱。组合的目标不是最大化收益，而是**最小化'所有策略同时亏损'的概率**。
